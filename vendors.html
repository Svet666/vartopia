Vartopia’s Saleforce package provides key enhancements to the traditional approach of treating deal registrations as leads.

The Vartopia package creates a deal registration object in Salesforce. Each deal registration is then linked to the following record types in Salesforce.

  • Partner Account
  • Customer Account
  • Customer Opportunity

Features Include:

  • Automated Partner System Activation
  • Support for Multiple Programs
  • Unique Deal Registration ID Numbers
  • Auto creation of Registration Expiration Date
  • Configurable Registration Statuses

All standard Saleforce functionality can be applied to the deal registration object including:

  • Workflow
  • Tasks & Notifications
  • Reporting and Dashboards
